Projects

Graphs Package

Created a Graph data structure implementing generic, depth- first, and breadth first traversals for the graphs in Java. Implemented an A* search algorithm for the graphs. Created a trip planning client for the package. Works with a Trip package that, when given a number of roads and locations, can find the shortest route to the given destination from the given origin.

Map Reduce Project

An AI, written in Java, that solves the game of connect four by using minimax to traverse the game tree of possible moves and find the best one. Uses Hadoop capabilities to traverse the game tree in parallel to significantly reduce the time it takes the AI to solve the game.

Image Recognizer

A program, written in C, which would take an image and tell user what digit the image is by comparing the image with a set of template images the program has stored. Would flip, rotate, transpose, and translate the template image onto the given image to determine the closest match between all the template images and the given image.
